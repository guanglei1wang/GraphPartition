Gravity

Mathematical Programming

Gravity is a c++ library for modeling and solving optimization problems.

It features a full-scale modeling langage, with optimized functions' evaluation and derivatives computing.

Gravity includes a mixed-integer nonlinear solver based on open-source code. It also links to various existing commerical and academic solvers.

Gravity features advanced optimization techniques such as smart on/off constraint modeling, automatic function convexification, outer-approximation and bounds propagation.

Created by Hassan Hijazi (hhijazi) in 2016/17
